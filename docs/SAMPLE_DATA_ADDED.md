## 🔄 Data Persistence

**Important:** This sample data is stored in your MySQL database. It will persist across application restarts. 

To reset and reload data:
1. Stop the application
2. Drop the database: `DROP DATABASE medicalstore_db;`
3. Recreate it: `CREATE DATABASE medicalstore_db;`
4. Restart the application - data will be auto-populated again
