## 🎨 UI Components Guide

### Stat Cards
```html
<div class="stat-card stat-primary">
    <div class="stat-icon">
        <i class="bi bi-icon"></i>
    </div>
    <div class="stat-content">
        <h6 class="stat-label">Label</h6>
        <h3 class="stat-value">Value</h3>
        <span class="stat-badge">Badge</span>
    </div>
</div>
```

**Variants:** stat-primary, stat-success, stat-warning, stat-info

### Modern Cards
```html
<div class="card modern-card">
    <div class="card-header">
        <h5 class="mb-0">
            <i class="bi bi-icon"></i> Title
        </h5>
    </div>
    <div class="card-body">
        Content
    </div>
</div>
```

### Empty States
```html
<div class="empty-state">
    <i class="bi bi-inbox"></i>
    <h5>Title</h5>
    <p>Description</p>
    <a href="#" class="btn btn-primary mt-2">
        <i class="bi bi-plus-circle"></i> Action
    </a>
</div>
```

### Page Headers
```html
<div class="page-header mb-4">
    <div>
        <h2>
            <i class="bi bi-icon text-primary"></i> Title
        </h2>
        <p class="text-muted mb-0">Description</p>
    </div>
    <a href="#" class="btn btn-primary">
        <i class="bi bi-plus-circle"></i> Action
    </a>
</div>
```

## 💡 Customization Tips

### Change Primary Color
Edit `layout.html` CSS section:
```css
:root {
    --primary-gradient: linear-gradient(135deg, YOUR_COLOR_1, YOUR_COLOR_2);
}
```

### Adjust Sidebar Width
```css
:root {
    --sidebar-width: 280px; /* Change from 260px */
}
```

### Modify Card Shadows
```css
.modern-card {
    box-shadow: 0 4px 16px rgba(0,0,0,0.1);
}
```

### Change Animation Speed
```css
.card, .stat-card {
    animation-duration: 0.7s;
}
```
